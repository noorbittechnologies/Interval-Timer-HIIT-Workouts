Interval Timer + HIIT Workouts

Fitness Interval Timer for all kind of Exercises & Trainings (Tabata).

  • Interval Timer + HIIT Workouts is easy to use and customisable setting all parameters (Time, Sound, Color and Name).
  • Is very useful for all kinds of interval trainings as you can have any kind of tasks with different name, sound and color with multiple repetition.
  • Works even in background mode.
  • Text to Speech for each Task Name with selective voice of your choice.
  • Create infinite numbers of Timers.
  • Calendar view for showing history.
  • Editable Timers.
  • Shuffle the tasks up and down.
